# Скрипты KengaCAD

| Скрипт | Назначение |
|--------|------------|
| `update_engine.py` | Обновление движка: `--build` (сборка из GoEngineKenga) или `--version 0.2.0` (загрузка с GitHub) |
| `build_release_exe.py` | Сборка exe |
| `build-engine.bat` / `build-engine.ps1` | Сборка движка из GoEngineKenga |
| `fix-pyqt6-dll.ps1` | Исправление DLL PyQt6 |

Файлы в корне: `update_engine.bat` — обновление движка (вызов update_engine.py).
