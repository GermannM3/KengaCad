# KengaCAD vs AutoCAD

## Сравнение

| | AutoCAD | KengaCAD |
|---|--------|----------|
| Сфера | Универсальный CAD | Траектории роботов, нанесение мастики |
| 2D | Полный | Линии, круги, дуги, полилинии, hatch, блоки, размеры |
| Редактирование | MOVE, COPY, STRETCH, BREAK, JOIN, EXPLODE... | Те же команды |
| Привязки | E, M, I, C, N, TAN, PER... | E, M, I, C, N |
| Форматы | DXF, DWG, PDF | DXF, DWG (ODA), PDF, .kengacad |
| 3D | Моделирование | Визуализация движка (робот, траектория) |
| Робототехника | Нет | Модель робота, траектория, симуляция, экспорт |

## Статус замены

| Область | Покрытие |
|---------|----------|
| 2D-черчение | ~95% |
| Редактирование | ~90% |
| Привязки | ~60% (NEA добавлена) |
| Форматы | ~80% |
| 3D | 0% |
| Робототехника | 100% |

Вывод: KengaCAD заменяет AutoCAD только для задачи 2D-чертёж траектории + робот + диспенсинг. Для общего черчения — частичная замена.

## Что осталось

- Привязки TAN, PER
- MTEXT, стили шрифтов
- Продвинутые размеры
- Настройка печати
