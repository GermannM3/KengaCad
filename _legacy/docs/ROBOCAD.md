# Совместимость с RoboCAD

KengaCAD поддерживает миграцию из RoboCAD:

- **Траектории:** CSV (X,Y,Z) — `TRAJECTORY file.csv`
- **Модель робота:** glTF/GLB
- **Команды:** SET_JOINT, LOAD_ROBOT, SIMULATE

Подробнее: ROBOCAD_ANALYSIS_GUIDE.md, ROBOCAD_FEATURES_ANALYSIS.md, ROBOCAD_COMMANDS_EXAMPLES.md
