KengaCAD v2.0.0 — Инструкция по установке

ТРЕБОВАНИЯ К СИСТЕМЕ
--------------------

Операционная система:
  - Windows 10/11 (64-bit)
  - Минимум: 4 ГБ ОЗУ
  - Рекомендуется: 8 ГБ ОЗУ

Процессор:
  - Intel Core i5 / AMD Ryzen 5 или выше

Видеокарта:
  - Поддержка OpenGL 3.3+
  - Минимум: 1 ГБ видеопамяти

Свободное место на диске:
  - Минимум: 500 МБ


УСТАНОВКА
---------

1. Запустите установщик KengaCAD_2.0.0_Setup.exe

2. Примите условия лицензионного соглашения

3. Выберите папку для установки (по умолчанию: C:\Program Files\KengaCAD)

4. Выберите дополнительные задачи:
   - Создать ярлык на рабочем столе
   - Создать ярлык в панели быстрого запуска

5. Нажмите "Установить"

6. Дождитесь завершения установки

7. Запустите KengaCAD из меню Пуск или с рабочего стола


ПЕРВЫЙ ЗАПУСК
-------------

1. При первом запуске KengaCAD автоматически проверит обновления

2. Рекомендуется настроить параметры в меню Сервис → Настройки

3. Для начала работы откройте справку (F1) или руководство ADVANCED_FEATURES.md


БЫСТРЫЙ СТАРТ
-------------

1. Нарисуйте полилинию:
   - Введите в командной строке: POLYLINE
   - Укажите точки мышью

2. Создайте траекторию:
   - Нажмите "Траектория из полилинии"
   - Или введите: TRAC_FROM_POLYLINE

3. Загрузите робота:
   - Перейдите на вкладку "Робот"
   - Нажмите "Загрузить робота"

4. Запустите симуляцию:
   - Нажмите "Симуляция"
   - Или введите: SIMULATE


УДАЛЕНИЕ
--------

1. Откройте Панель управления → Программы и компоненты

2. Найдите "KengaCAD" в списке

3. Нажмите "Удалить"

4. Подтвердите удаление


ТЕХНИЧЕСКАЯ ПОДДЕРЖКА
---------------------

Email: support@kengacad.local
Документация: ADVANCED_FEATURES.md, COMMAND_REFERENCE.md


ИЗВЕСТНЫЕ ОГРАНИЧЕНИЯ
---------------------

1. Для работы с STEP/IGES файлами требуется отдельная установка:
   pip install cadquery

2. Для расширенных 3D-функций рекомендуется установить:
   pip install pyvista pyvistaqt trimesh scipy


ОБНОВЛЕНИЕ
----------

Программа автоматически проверяет обновления при запуске.
Также можно проверить вручную: Справка → Проверить обновления.
