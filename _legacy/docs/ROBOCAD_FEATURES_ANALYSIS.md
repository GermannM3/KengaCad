# Анализ RoboCAD и возможные заимствования для KengaCAD

## Обзор RoboCAD

RoboCAD - это профессиональное CAD-приложение для программирования и симуляции роботов. Оно предоставляет мощные инструменты для:

- Программирования траекторий роботов
- Симуляции движения
- Визуализации процессов
- Интеграции с различными робототехническими платформами

## Возможные заимствования для KengaCAD

### 1. Форматы файлов

#### Траектории
RoboCAD, вероятно, использует специфические форматы для хранения траекторий:
- Точки траектории (XYZ координаты)
- Углы суставов
- Скорости и ускорения
- Временные метки
- Параметры процесса (например, для диспенсинга)

**Для KengaCAD**: Реализовать поддержку импорта/экспорта траекторий в форматах RoboCAD

#### Роботы
Форматы определения роботов:
- Кинематическая цепь
- Параметры Денавита-Хартенберга
