# KengaCAD vs RoboCAD (Siemens Tecnomatix)

## Что есть в RoboCAD

- 3D workcell — единая среда: робот, оснастка, траектории, столы
- Kinematic models — полная кинематика, 50+ моделей из коробки
- Path planning — создание и редактирование траекторий в 3D
- Collision detection — mesh-to-mesh проверка коллизий
- Offline programming — экспорт программ для KUKA, ABB, Fanuc и т.п.
- Virtual commissioning — симуляция с PLC-кодом, Gantt-диаграммы цикла

## Чего пока нет (возможные доработки)

- Полноценный 3D picking (gizmo translate/rotate)
- Импорт CATIA V5 / SolidWorks нативный (STEP достаточно)
- Автоматическая оптимизация траектории (обход препятствий)
- Расчёт энергопотребления и нагрузок
- Profinet / EtherCAT прямая интеграция (через OPC UA)

## Порядок демо для роботиста

1. Запустить KengaCAD, показать split view (2D + 3D)
2. Выбрать робот из библиотеки (KR 210 R2700 или IRB 6700)
3. Подвигать оси в панели j1-j6, показать wireframe-робот
4. Ввести координаты TCP — IK автоматически посчитает углы
5. Добавить объекты workcell: стол, оснастку, ограждение
6. Загрузить STEP-геометрию (если есть)
7. Построить траекторию, проверить коллизии
8. Экспортировать в KRL / RAPID / TP / INFORM / URScript
9. Показать настраиваемые шаблоны постпроцессоров
10. Открыть PLC / Сигналы I/O — показать таблицу сигналов, привязку к траектории
11. Открыть Virtual Commissioning — сгенерировать цикл, показать Gantt-диаграмму
12. Запустить симуляцию цикла — показать синхронизацию сигналов с шагами
