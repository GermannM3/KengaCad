# Чеклист перед релизом KengaCAD Professional

Автотестов в репозитории для UI нет; перед выкладкой установщика пройдите ручную проверку на чистой Windows 10/11 x64 (без отдельно установленного .NET — так проверяется self-contained).

## Сборка

1. `.\build_installer_professional.ps1` — без ошибок.
2. В логе публикации: **~150–200 МБ**, **200+ файлов**, есть `KengaCAD.exe` и `clrjit.dll` в корне `publish/`.
3. `installers\Output\KengaCAD_Professional_Setup.exe` — **~50–60 МБ** (сжатый LZMA).

## Установка и запуск

4. Установка в профиль без прав администратора или с UAC — приложение стартует.
5. Окно открывается, сетка/риббон видны, холст реагирует на мышь.

## Файл и правка

6. **Новый / Открыть / Сохранить** (Ctrl+N, Ctrl+O, Ctrl+S).
7. **Отмена / Повтор** (Ctrl+Z, Ctrl+Y).
8. **Удалить** выбранное (Delete).

## Чертёж 2D

9. Линия, круг, дуга, полилиния, прямоугольник.
10. Переместить, копировать, повернуть, масштаб, зеркало.
11. Обрезать, удлинить, скругление (по возможности на простой геометрии).
12. Новый слой; сетка / орто / привязка (чекбоксы).

## Вид

13. Показать всё, зум +/-, панорама.
14. Виды 3D: сверху, спереди, слева, 3D, изометрия (кнопки в панели 3D).

## Робот и траектория

15. Загрузить / демо-робот; «В ноль».
16. Траектория: из полилинии, сплайн, сгладить, спираль.
17. Симуляция: старт, пауза, стоп, сброс.

## Экспорт

18. G-код, KRL, RAPID, Fanuc TP, Yaskawa, UR Script — диалог сохранения, файл создаётся без исключения (содержимое на разумность).

## Импорт и прочее

19. Импорт STEP/IGES/STL/glTF — по наличию тестовых файлов (допустимо «не реализовано» без падения, если так задумано).
20. Блок: создать / вставить; текст; линейный размер / радиус.

## Сбои

21. При аварии проверить `%LocalAppData%\KengaCAD\crash_log.txt`.

Отмечайте пункты по мере проверки. Блокирующие для релиза: п. 1–6, 9, 13–17, 18 (хотя бы один постпроцессор).
