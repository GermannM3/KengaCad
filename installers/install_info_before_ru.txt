KengaCAD Professional — автономная установка

В состав установки входит среда выполнения .NET 8 и графические библиотеки (включая SkiaSharp). Отдельно устанавливать .NET Runtime не нужно.

Файл установщика сжат (LZMA): размер загрузки меньше, чем занятое на диске место после установки (обычно порядка 150–200 МБ в папке программы).

Если Windows блокирует запуск: у неподписанных программ «Интеллектуальное управление приложениями» может запрещать установку. Для релиза нужна код-подпись издателя; для проверки можно использовать портативный ZIP из той же сборки или временно ослабить политику (см. docs\WINDOWS_TRUST_AND_SIGNING.md).

Первый запуск может занять несколько секунд из-за инициализации среды.

Требования: Windows 10/11, 64-bit.
